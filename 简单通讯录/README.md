# 通讯录

简单地实现通讯录功能，支持增删改查

## Compile

```shell
$ g++ contacts.cpp -o contacts -std=c++11
```

## Example

```shell
$ ./contacts
        --- 通讯录 ---
联系人初始化为空
----------------
选择功能：
        1. 添加联系人
        2. 查找号码
        3. 联系人列表
        4. 联系人数据
        5. 删除联系人
        6. 保存联系人
        7. 退出系统
>>
```

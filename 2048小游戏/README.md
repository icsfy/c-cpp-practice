# 2048

通过 ncurses 库实现的终端下的 2048 小游戏。

## 编译运行

```shell
$ g++ -o 2048 2048.c -lncurses
$ ./2048
```

## 游戏说明

可以通过上下左右或者 `w` `s` `a` `d` 按键进行控制

`r` 键可以重新开始游戏, `q` 键退出游戏

游戏默认 `4x4` 方格, 可以通过命令行参数指定 `nxn` 的方格

例如

```shell

                                            score: 0
                            ┌─────┬─────┬─────┬─────┐

                            ├─────┼─────┼─────┼─────┤
                            │     │     │     │     │
                            │     │     │     │     │
                            │     │     │     │     │
                            ├─────┼─────┼─────┼─────┤
                            │     │     │     │     │
                            │     │  4  │     │     │
                            │     │     │     │     │
                            ├─────┼─────┼─────┼─────┤
                            │     │     │     │     │
                            │     │     │     │     │
                            │     │     │     │     │
                            └─────┴─────┴─────┴─────┘


                            'r': Restart game
                            'q': Quit game
```
